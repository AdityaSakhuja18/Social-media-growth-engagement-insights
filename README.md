# 📊 Social Media Growth & Engagement Insights

## 🚀 About This Project

This project explores how social media content performs across different platforms by analyzing engagement patterns such as likes, comments, shares, and overall interaction trends.

The goal is to understand what type of content works best, which platforms drive the most engagement, and how user interaction changes over time.

The project follows a complete analytics workflow including **data cleaning in Python, SQL-based validation, and dashboard creation in Power BI**.

---

## 📂 Dataset Overview

- Total Records: 10,000+
- Data Source: Social Media Performance Dataset
- Covers multiple platforms including YouTube, Instagram, LinkedIn, and Medium

### Key Columns:
- Post engagement metrics (likes, comments, shares)
- Platform details
- Content type (video, image, text, reel)
- Posting date
- Regional engagement data

---

## 🛠️ Tools Used

- Python (Pandas, NumPy) – Data cleaning and analysis  
- MySQL – Data validation and querying  
- Power BI – Dashboard and visualization  
- DAX – KPI calculations  
- Excel – Initial data inspection  

---

## 🎯 Objective

The main objective of this project is to identify:
- What drives higher engagement on social media
- Which platforms perform best in terms of reach and interaction
- Which content formats attract the most user attention
- How engagement changes over time and across regions

---

## 🔍 Workflow Breakdown

### 1. Data Preparation (Python)
The dataset was cleaned and structured using Python. Missing values were handled, duplicates were removed, and a new metric called **Total Engagement** was created:

**Total Engagement = Likes + Comments + Shares**

---

### 2. Data Validation (SQL)
SQL queries were used to validate the processed data and extract insights such as:
- Platform-wise engagement comparison
- Top-performing posts
- Overall engagement distribution

---

### 3. Data Analysis (EDA)
Exploratory analysis helped identify:
- Which platforms generate the most engagement
- Which content types perform consistently better
- Patterns in user interaction behavior

---

### 4. Dashboard Development (Power BI)
An interactive dashboard was built with:
- KPI cards for quick insights
- Filters for platform, content type, and time period
- Visual charts showing engagement trends
- Drill-down capability for deeper analysis

---

## 📌 Key Insights

- Video content consistently performs better than other formats  
- YouTube leads in overall reach, while Instagram shows strong engagement rates  
- Engagement varies significantly based on content type and timing  
- Certain months show noticeably higher interaction levels
